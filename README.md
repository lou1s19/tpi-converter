# TPI Converter

A simple and lightweight conversion tool developed for **TPI Media**.

The application provides a fast and intuitive way to perform conversions through a clean and responsive interface.

## Features

- ⚡ Fast calculations
- 📱 Responsive design
- 🎨 Simple and intuitive user interface
- 🔧 Lightweight and easy to maintain

## Tech Stack

- HTML
- CSS
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/lou1s19/tpi-converter.git
```

Install dependencies (if required):

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## About

This project was developed as part of my work at **TPI Media**. It is designed to provide a fast, reliable, and user-friendly conversion tool for everyday use.
